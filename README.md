# Pho-dog (“photo” + “dog”)

## 프로젝트 내용
>**강아지 심박수를 이용한 아두이노 카메라**   
>**개발기간 : 2022 - 03 - 02 ~ 2022 - 06 - 27**

<hr></hr>

## 프로젝트 소개
심장박동센서와 카메라를 아두이노로 연동하고 강아지에게 센서를 부착하면 강아지가 좋아하는 것을 보고 
심장박동 수가 높아질 때  스스로 사진을 찍어 강아지의 취향을 알아내고 귀여운 사진도 남길 수 있는 장치
